# billbird-v1

Build Billbird v1: a lightweight, self-hostable time tracking tool for GitHub-native software teams. Developers log hours via slash commands in issue comments (/log, /correct). A GitHub App bot acknowledges entries and stores them in Postgres. Supports client attribution via GitHub labels, non-destructive correction chains, passive cycle-time tracking from project board movements, and a minimal HTMX admin panel with GitHub OAuth. Deployable via Docker Compose and Helm.
