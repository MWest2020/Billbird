# Billbird documentation

Billbird turns GitHub issue comments (`/log`, `/plan`, and their corrections)
into an auditable, label-attributed time-tracking dataset. It is API-first,
self-hosted, and single-tenant (one instance per organisation). For the
project overview, quick start, and platform-support notes, see the
[README](../README.md).
